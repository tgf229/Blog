###-定时器
－ES6中timer-mixin的写法已经被timer取代，此方法可实现延时或定时器，如：
```javascript
setToggleTimeout(){
    this.timer = setTimeout(
        ()=>{
            console.log('延时器..')
            this.setState({animating: !this.state.animating});
            this.setToggleTimeout();
        },1000
    )
}
```
需记住在 componentWillUnmount()的时候执行
```javascript
    this.timer && clearTimeout(this.timer);
```

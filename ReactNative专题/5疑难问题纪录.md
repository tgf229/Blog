###-this传递
render内部引用其他组件的时候，this就会变为此组件，解决方案为 render的时候 将this赋值，然后在第三方组件内使用被赋值后的this。 
代码：
```javascript
render(){
    var bannerThis = this;
    return(
        <Swiper autoplay={true} loop={true}>
            {this.state.data.map(function(item){
                return(
                    <View key={item.id} >
                    <TouchableHighlight
                        onPress={()=>bannerThis.onBannerClick()}>
                    <Text>{item.name}</Text>
                    </TouchableHighlight>
                    </View>
                    )
            })}
        </Swiper>
    )
}
```
